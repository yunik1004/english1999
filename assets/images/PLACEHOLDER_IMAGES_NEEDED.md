# Placeholder Images Required

The hierarchical navigation system requires poster images for chapters and subchapters.

## Required Images:

### Chapter Posters (2:3 aspect ratio)
- `chapters/chapter_1.png` - 400x600px minimum

### SubChapter Posters (2:3 aspect ratio)  
- `subchapters/1_1.png` - 400x600px minimum

## Specifications:
- **Format**: PNG (lossless, supports transparency)
- **Size**: 400x600px minimum (2:3 aspect ratio)
- **Style**: Should match Reverse: 1999 game aesthetic

## Temporary Solution:
For testing purposes, you can use any image temporarily. The app will show a broken image icon if images are missing.

To add your images:
1. Place PNG files in the respective directories
2. Restart the app (run `flutter clean && flutter run`)
