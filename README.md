# English: 1999

Learn English through the story of [Reverse: 1999](https://re1999.bluepoch.com/).

**Live**: [yunik1004.github.io/english1999](https://yunik1004.github.io/english1999/)

## Features

- Watch story chapters with synchronized transcriptions
- Toggle between video and audio-only mode
- Show/hide English translations
- Supports web and Android

## Tech Stack

- Flutter (Web + Android)
- YouTube IFrame API (`youtube_player_iframe`)
- `go_router` for navigation
- `provider` for state management
- Hosted on GitHub Pages

## Development

```bash
# Run on web
flutter run -d chrome

# Build Android APK
flutter build apk --release

# Build and deploy web
flutter build web --release
```
